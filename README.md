This is Legion as it was meant to be. There are no hard edges. It is perfectly smooth in all directions.

Use this tweak: `!bset tweakdefs <copy and paste contents of encoded.txt>`

### What it does

Removes every single faction-exclusive mechanic, idea, and design from Legion.

### Why would you do that

So that finally Gamers can be free. Legion cannot hurt them anymore because it contains zero Legion.

### Known issues

Many things in BAR cannot be modified through tweaks. These are generally limits within the unit scripts. However, this is not an exclusive trait of Legion, so is not targeted for removal by this tweak.

- The Mantis, Gelasma, and Telchine are not addressed yet. Each has a new "thing" coming to it.
- The Syracusia (destroyer) cannot fire its depth charges. It may need an anti-sub drone, but this is not Smooth.
- The Medusa (heavy missile vehicle) cannot fire its weapons independently and has a scripted/forced reload period.
- Railguns still have their CEG trail effect that follows after the projectile.
- The Phobos has a huge area of effect.
- Non-cluster plasma looks way too cool which is not very Smooth.
- Some units are still using LaserCannon-based machine guns which is not very Smooth.
- Some gatling units especially need their owner accuracy weights reduced by half and some spray angle moved into accuracy.
