This is Legion as it was meant to be: perfectly smooth in all directions.

### Use this tweak

Copy the contents of `tweakdef_minified_encoding.txt` into the tweakdefs field in your lobby.

### What it does

Removes every single faction-exclusive mechanic, idea, and design from Legion.

You can read the detailed change list in the [gist](gist.md).

### Why

So that finally Gamers can be free. Legion cannot hurt them anymore because it contains zero Legion.

### Known issues

- The code is increasingly deranged to fit under 12k characters (the limit is 16k).
